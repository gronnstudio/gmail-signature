# GRØNN signature assets

Deze repo is bedoeld om assets voor je e-mailhandtekening publiek te hosten.

## Structuur

- `assets/logo-vierkant.jpg` — hoofdlogo
- `assets/icons/instagram-icon.png` — Instagram icoon
- `assets/icons/whatsapp-icon.png` — WhatsApp icoon
- `examples/gmail-signature-template.html` — voorbeeld HTML met GitHub raw placeholders

## GitHub raw URLs

Vervang `JOUW_GITHUB_NAAM` en eventueel `main` als je branch anders heet.

- `https://raw.githubusercontent.com/JOUW_GITHUB_NAAM/gronn-signature-repo/main/assets/logo-vierkant.jpg`
- `https://raw.githubusercontent.com/JOUW_GITHUB_NAAM/gronn-signature-repo/main/assets/icons/instagram-icon.png`
- `https://raw.githubusercontent.com/JOUW_GITHUB_NAAM/gronn-signature-repo/main/assets/icons/whatsapp-icon.png`

## Werkwijze

1. Upload je logo en iconen naar de juiste map.
2. Maak de repo public.
3. Controleer of de raw links direct openen in de browser.
4. Gebruik die links in je Gmail-signature of in de HTML-template.

## Tip

Voor productie is een eigen domein zoals `https://gronn.studio/assets/...` nog netter, maar GitHub raw werkt prima om te starten.
